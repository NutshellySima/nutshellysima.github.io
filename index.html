What I build

I focus on systems that make large models practical in production: freshness, cost, reliability, and performance.

Ekko (WeChat recsys)

Low-latency model updates at extreme scale

  • Compressed dissemination + model-aware scheduling (WAN −92%)
  • SLO-aware placement (machine cost −49%) + safe rollback
  • Second-level freshness (2.4s) for multi-terabyte models

More context (optional) Open if you want the longer story

A recurring theme in my work is system–algorithm co-design: scaling model size can improve offline metrics, but production outcomes depend on freshness, resource scheduling, and safe deployment mechanisms.

Recently, I’m also interested in cost-effective LLM serving (e.g., remote KV-cache storage and compression) and how workload characteristics shape systems design.
